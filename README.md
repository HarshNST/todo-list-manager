# To-Do List Manager

## Project Description
This is a simple to-do list app where users can add and manage their tasks. It will also show some tasks from an API. I will add features like search, filter, and sort.

## Features
. Add tasks
. Delete tasks
.Mark tasks as done
.Search and filter tasks
